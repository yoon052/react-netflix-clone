# 넷플릭스 클론 웹사이트
[넷플릭스 클론](https://yoon052.github.io/react-netflix-clone/)

<br/>

넷플릭스 클론 웹사이트를 구현했습니다. <br/>
메인 페이지, 상세 페이지, 다음으로 이동하는 버튼, 검색 창, 검색 결과 페이지 등 넷플릭스와 비슷하게 기능을 구현했습니다.

<br/><br/>


## 구현 기능
- 행별로 넷플릭스 오리지널, 현재 인기 작품, 높은 평점을 받은 작품, 액션 영화, 코미디 영화 로 분류해 작품들의 썸네일을 가로로 배치했습니다.
- 좌우 양끝의 화살표 버튼을 누르면 각 행에 해당하는 작품들을 추가로 확인할 수 있습니다.
- 우측 하단의 원형 버튼을 누르면 행의 원하는 작품 배열 순서로 바로 넘어갈 수 있습니다.
- 썸네일 위에 마우스를 올리면 썸네일이 팝업되는 기능을 구현했습니다.
- 썸네일을 누르면 나에게 추천하는 작품에 부합하는지 나타내는 퍼센테이지와 개봉 날짜, 평점, 작품의 대략적인 줄거리를 담고 있는 상세 페이지를 구현했습니다.
- 메인 페이지 가장 상단의 작품에서 Play 버튼을 클릭하면 해당 작품의 유튜브 예고편이 재생됩니다.
- 영화 검색창에서 검색을 하면 검색어와 관련된 작품들이 결과창에 뜨는데, 작품을 누르면 썸네일이 상세 페이지에 그대로 구현됩니다.
- 반응형 웹사이트여서 페이지 크기를 조절하면, 페이지의 컨텐츠 수와 크기도 함께 조정됩니다.
